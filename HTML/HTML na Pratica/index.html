Essa feramente já vem instalado no seu navegador. Utilizamos essa ferramenta para fazer análise e estudo de códigos nos sites que visitamos. Por essa ferramenta conseguimos ver como está formatado o HTML da página e o CSS. E conseguimos fazer algumas alterações na página para efeito de estudos.

Estrutura básica do HTML

inicio

Todo arquivo HTML consiste de 3 partes Html, Head e Body Html, serve para escrevermos todo o código. Head, são códigos que vão ser carregados, porém não aparece na página Body, escrevemos aqui tudo que irá aparecer na nossa página.

Principais Tags usados no Html

inicio

A maioria das Tags tem uma abertura e um fechamento. Tag P serve para criarmos uma paragrafo Tag Li serve para criarmos uma linha. Tag H1 serve para criarmos título Tag H2 serve para criarmos sub título Tag B deixa o texto em negrito Tag I deixa o texto em itálico Tag U deixa o texto sublinhado. Porem temos algumas Tegs que não tem fechamento como a s tags imput e Img

Atributos de Tags

inicio

Atributos nada mais são doque uma propriedade que a tag ira receber Tag Id, serve para identificar os textos que queremos mexer Tag Style, server para formatarmos o texto selecionado. Tag Class, serve para identificar e alterar textos já identificados. Tag Imput, serve para criar caixas de texto, cores e etc Tag Img, serve para colocar imagens arquivos ou links de imagens.

Listas ordenadas e listas não ordenadas

inicio

Para criarmos listas utilizamos as tags OL e IL Tag OL cria uma lista ordenada, o navegador insere números nas linhas Tag UL cria uma lista não ordenada. O navegador insere marcadores nas linhas.

Links

inicio

Para colocarmos uma link no código usarmos a Tag A Quando clicamos no no texto com link, a nova pagina em questão sera aberta na mesma pagina. Caso queira mudar isso usamos o Target _ blank Também temos um atributo chamado Title, ele cria uma balão com uma escrita quando colocamos o mouse sobre o texto com o link. Isso serve para chamar atenção do usuário ou explicar a onde o link ira de levar.
